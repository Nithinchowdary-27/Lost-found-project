# 🔍 Lost & Found Management System

A web-based application to manage lost and found items efficiently in colleges and public places.

---

## 📌 Features

- 👤 User Registration & Login
- 📦 Add Lost Items
- 📦 Add Found Items
- 🖼 Image Upload
- 📊 Dashboard View
- 🔴 Lost & 🟢 Found Separation
- ❌ Delete Items

---

## 🛠 Tech Stack

- **Frontend:** HTML, CSS  
- **Backend:** Python (Flask)  
- **Database:** JSON (data.json, users.json)  
- **Storage:** Local uploads folder
